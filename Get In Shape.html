Get In Shape

Intervals

Intervals are important. Intervals can help us get in shape. An interval is where for about 3 minutes you sprint. Then, after those 3 minutes you slow to a jog. Next, you do another 3 minutes and you may want to walk or sprint again. It just depends of what you want to do.

What Should I Run First?

  • Start off slow. You may want to run fast, but don't. You actually need to run slower than you think you can.
  • Don't run super far. For your first run I would run about a 1/4 of a mile. 1/4 of a mile is 1 lap around the track.
  • And run more often. Each day run a little faster and a little farther. The more you run, the better you will get. You will build up your endurance.
  • And of course, stretch before you run. This way, you won't pull any muscles.
